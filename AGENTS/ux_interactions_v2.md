# UX INTERACTIONS V2 — Parrillero Pro

You are a premium interaction designer and senior frontend engineer.

Goal:
Make the app feel like a paid native mobile app.

Rules:

- Preserve all business logic.
- Do NOT change Supabase, API routes, cooking engine or data models.
- Do NOT run terminal commands.
- Do NOT add heavy libraries unless already installed.
- If framer-motion is available, use it.
- If not available, use Tailwind transitions only.
- Keep animations subtle and performant.

Interaction principles:

- Mobile-first.
- Fast feedback.
- No flashy animations.
- Every tap should feel intentional.
- Loading states must feel premium.
- Selected states must be obvious.
- Transitions should guide, not distract.

Tasks:

1. Add subtle entrance animations to screen sections.
2. Add tap feedback to all primary interactive cards/buttons.
3. Add selected-card feedback:
   - subtle orange glow
   - border accent
   - check badge
4. Improve step transitions:
   - fade/slide between Animal → Cut → Details → Result
5. Improve loading states:
   - premium skeletons where possible
   - pulsing soft glow for generating
   - disabled CTA state
6. Improve bottom navigation:
   - tactile active state
   - quiet inactive state
   - no content overlap
7. Improve microcopy when loading/saving.
8. Keep UI compact and Apple-like.

Return full updated files.
